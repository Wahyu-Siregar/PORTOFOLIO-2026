# Brutalist Portfolio Website

Portfolio website for Wahyu Muliadi Siregar - Computer Science Student specializing in Network Security, Cybersecurity, and Digital Forensics.

## Features

- **Pure HTML & CSS**: No frameworks, no JavaScript
- **Brutalist Design**: Raw, functional, honest aesthetic
- **Lightweight**: Total page size < 100KB
- **Fast Loading**: < 1 second on 3G
- **Responsive**: Works on all devices
- **Accessible**: Semantic HTML5, works without JavaScript

## Structure

```
.
├── index.html      # Main portfolio page
├── style.css       # Minimal CSS stylesheet (<5KB)
├── assets/         # Optimized images (optional)
└── README.md       # This file
```

## Technical Stack

- HTML5
- CSS3 (Vanilla)
- No JavaScript
- No external dependencies
- System fonts only

## Design Principles

1. Content first, design second
2. Visible structure and borders
3. Monochrome color palette
4. Grid-based box layout
5. Left-aligned text
6. No rounded corners
7. Minimal CSS

## Performance

- Total HTML + CSS: ~20KB
- No external dependencies
- Works with JavaScript disabled
- No web fonts loaded
- Loads in < 1 second

## Local Development

Simply open `index.html` in any web browser. No build process required.
